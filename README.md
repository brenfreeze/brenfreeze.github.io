# brenfreeze.github.io

Personal portfolio of Bren Aviador — an editorial one-pager built with Vite + React + TypeScript, deployed to GitHub Pages via Actions.

- `pnpm dev` — local dev server
- `pnpm build` — typecheck + production build to `dist/`
- `pnpm test` — vitest unit tests

Swap the headshot by replacing `public/headshot.jpg`; the ASCII portrait regenerates at runtime.
